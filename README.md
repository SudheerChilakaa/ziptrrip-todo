# Ziptrrip Todo Application

## Overview
A full-stack Todo application built using the MERN stack.

## Features

- Create Todo
- View Todos
- Update Todo
- Delete Todo
- Search Todos
- Filter by Status
- Filter by Priority
- Sort by Due Date
- Sort by Priority
- Pagination
- Loading Spinner
- Toast Notifications
- Responsive UI

## Tech Stack

Frontend
- React
- Vite
- Axios
- React Router
- React Toastify

Backend
- Node.js
- Express.js
- MongoDB
- Mongoose

## Folder Structure

backend/
frontend/

## API Endpoints

POST /api/todos

GET /api/todos

GET /api/todos/:id

PUT /api/todos/:id

DELETE /api/todos/:id

## Installation

### Backend

cd backend

npm install

npm run dev

### Frontend

cd frontend

npm install

npm run dev
